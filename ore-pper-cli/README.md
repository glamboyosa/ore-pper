# Ore-pper CLI

The CLI for the `<Stepper/>` component. It writes the component to the user's React web app.

## Usage

```bash
npx ore-pper-cli
# or
pnpm dlx ore-pper-cli
```

Documentation can be found [here](https://ore-pper.glamboyosa.xyz)

## Tools

This CLI is built with:

- TypeScript
- Node.js
- Inquirer.js
